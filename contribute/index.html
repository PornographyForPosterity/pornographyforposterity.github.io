Contribute - Sun, Jul 12, 2026

How to submit corrections, sources, or new entries to the catalog.

PFP is maintained by volunteers and the record it’s trying to keep is much larger than what’s catalogued so far. Useful contributions include:

  • Corrections — a wrong date, a misattributed alias, a studio credit that doesn’t match the source material.
  • Sources — trade press, box art, official studio releases, or other public documentation that would support a new or existing entry.
  • New entries — performers or studios not yet catalogued, especially from the project’s initial pre-digital-era focus (see History & Scope Notes).
  • Performer requests — corrections or removal requests from a performer or their verified representative about their own entry.

How to submit

This site is built from the pornographyforposterity.github.io repository on GitHub. The most reliable way to contribute is to open an issue or pull request there with:

  1. What you’re adding or correcting, and for which entry.
  2. A citation — a link or clear reference to the public source backing the change.

If you’re not comfortable using GitHub directly, a plain description of the correction and its source is still useful — reach out through the repository’s issue tracker and it will get transcribed.

What we can’t accept

See Methodology: Scope & Ethics for what’s permanently out of scope — this project doesn’t catalog illegal content, minors, non-consensual material, or performers’ private (non-professional) information, and can’t accept submissions of that kind.
